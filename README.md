# Optimal Course Scheduler

Full-stack decision-support system for selecting and scheduling academic courses using dynamic programming, Pareto trade-off analysis, and schedule packing.

## Stack

- Backend API: Python standard library HTTP server
- Optimization engine: C++ with Python fallback
- Frontend: Static single-page app with modular JavaScript

## Run

```powershell
py run.py
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000)

## Smoke Test

```powershell
py backend\tests.py
```
